# Dashboard-de-Vendas-do-Xbox-com-Excel-DIO

🎮 Xbox Game Pass Subscriptions Sales Dashboard
🎯 Objetivo do Projeto

Este projeto tem como objetivo a criação de um dashboard de vendas de assinaturas, focado na organização, análise e visualização de dados relacionados aos serviços de assinatura da Microsoft e parceiros, como Xbox Game Pass, EA Play Season Pass e Minecraft Season Pass.

A proposta é transformar dados brutos de vendas em informações visuais claras e estratégicas, facilitando a análise de desempenho e apoiando a tomada de decisões orientadas por dados.

📊 Visão Geral do Dashboard

O dashboard apresenta uma visão consolidada das vendas de assinaturas no período analisado, permitindo acompanhar:

Receita total por tipo de assinatura

Comparação entre diferentes produtos

Distribuição das assinaturas do Xbox Game Pass

Filtros por tipo de plano

Período de cálculo: 01/01/2024 a 31/12/2024
Última atualização: 25/12/2024 às 09:00
